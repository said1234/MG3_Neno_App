Die Sound-Ausgaben laufen auf praktisch jeder Emulator-Konfiguration.


Für die Bild- und Videoanzeigen muss der Emulator Zugriff auf eine simulierte SD-Karte haben.

1. Rufen Sie dazu den Befehl Run/Run Configurations auf.
2. wählen Sie im Dialog Run Configurations ggf. links unter Android Application 
   Ihre App aus und wechseln Sie rechts zur Registerseite Target.
3. Wählen Sie nun entweder die Option Automatic und anschließend ein AVD mit SD-Karte 
   oder geben Sie den Namen einer zuvor angelegten Imagedatei unter "Additional
   Emulator Command Line Options" ein.

   Mehr zu diesen Schritten in Kapitel 19.1 und Anhang C.

4. Klicken Sie auf Run.



Achtung, die Videoausgabe wird nur bei Ausführung auf einem Smartphone erscheinen.
